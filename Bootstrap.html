HTML Perusteet

Tällä sivulla esitellään HTML:n perusrakenne ja yleisimmät tagit.

<HTML>

HTML-tagi määrittää koko HTML-dokumentin rakenteen.

<HEAD>

HEAD-osio sisältää metatiedot, kuten otsikon ja linkit tyylitiedostoihin.

<BODY>

BODY-osio sisältää sivun varsinaisen sisällön.

<P>

P-tagi luo kappaleen.

<IMG>

IMG-tagi lisää kuvan. Esimerkki:

<img src="kuva.jpg" alt="Kuvaus">
<A>

A-tagi luo linkin. Esimerkki:

<a href="https://example.com">Linkki</a>

CSS ja Bootstrap

CSS:llä voidaan muotoilla sivun eri osia. Esimerkiksi:

body { background-color: #f8f9fa; }

Bootstrap tarjoaa valmiita komponentteja, kuten navigointipalkkeja ja kortteja, jotka helpottavat kehitystä.
